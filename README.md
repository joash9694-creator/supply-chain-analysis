# 📦 Supply Chain Analytics Dashboard

## 🔹 Project Overview
This project demonstrates end-to-end Supply Chain Analytics using SQL, Python, and Power BI.

## 🔹 Business Requirement
Supply chain companies often face challenges such as inventory shortages, delayed deliveries, high logistics costs, and inefficient supplier performance. This project helps analyze supply chain operations and provides insights to improve efficiency, reduce costs, and support better business decisions.

## 🔹 Tools Used
- SQL (MySQL)
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## 🔹 Key Analysis
- Inventory Performance Analysis
- Supplier Performance Analysis
- Sales & Profit Analysis
- Regional Performance Analysis
- Delivery & Logistics Analysis
- Monthly Sales Trends

## 🔹 Insights
- Identified top-performing regions based on revenue.
- Analyzed supplier performance and delivery efficiency.
- Monitored inventory levels to reduce stock shortages.
- Evaluated profit contribution by product categories.
- Tracked monthly sales and operational trends.
