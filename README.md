# supply-chain-analysis
📦 Supply Chain Analytics Dashboard
🔹 Project Overview

This project focuses on Supply Chain Analytics using SQL, Python, and Power BI. The objective is to analyze procurement, inventory, sales, logistics, and supplier performance to improve operational efficiency and support data-driven decision-making.

🔹 Problem Statement

Supply chain operations generate large volumes of data from suppliers, warehouses, transportation, and customers. This project analyzes supply chain performance to identify bottlenecks, optimize inventory levels, reduce costs, and improve delivery efficiency.
